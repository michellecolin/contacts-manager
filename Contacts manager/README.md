# contacts manager

### Install, Build, Run

### install
```
typings install
npm install
```

### Run
```
gulp
```
### Build dist

```
gulp build
```

### Run dist
You can run the dist using the http-server npm module
```
npm install http-server -g
```
Go to the dist folder and run the server
````
cd dist
http-server
````

Access the project through the provided url.
